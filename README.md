## 🛠️ Tecnologie Utilizzate
- **Frontend**: React, Next.js, Expo, Tailwind CSS, Framer Motion.
- **Backend & DB**: Supabase, Prisma (PostgreSQL), Socket.io, Redis.
- **Tools**: TypeScript, Vite, Resend.

## 📦 Come Avviare Localmente
Il repository è configurato per avviare tutti i progetti contemporaneamente tramite lo script centrale nel `central-hub`.

1.  Entra nella cartella `central-hub`:
    ```bash
    cd central-hub
    ```
2.  Installa le dipendenze:
    ```bash
    npm install
    ```
3.  Avvia tutti i server di sviluppo:
    ```bash
    npm run dev
    ```
