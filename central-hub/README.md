# Central Hub

Menu centrale per accedere a tutti i tuoi progetti web.

## Progetti Disponibili

1. **Climate Action Hub** (Porta 3001) - Piattaforma per azioni climatiche
2. **Meme Generator** (Porta 3002) - Creatore di meme  
3. **My Digital Diary** (Porta 3003) - Diario personale
4. **Polyglot Translator** (Porta 3004) - Traduttore multilingua

## Avvio Rapido

### Avvia tutti i progetti con un solo comando:
```bash
cd central-hub
npm install
npm run dev
```

Questo comando avvierà automaticamente:
- Central Hub (Porta 3000)
- Climate Action Hub (Porta 3001)  
- Meme Generator (Porta 3002)
- My Digital Diary (Porta 3003)
- Polyglot Translator (Porta 3004)

### Avvio manuale dei singoli progetti:

Se preferisci avviare i progetti singolarmente:

```bash
# Central Hub
cd central-hub
npm run dev:hub

# Climate Action Hub
cd climate-action-hub
npm run dev

# Meme Generator  
cd meme-generator
npm run dev

# My Digital Diary
cd my-digital-diary
npm run dev

# Polyglot Translator
cd polyglot-translator
npm run dev
```

## Accesso

- **Central Hub**: http://localhost:3000
- **Climate Action Hub**: http://localhost:3001
- **Meme Generator**: http://localhost:3002
- **My Digital Diary**: http://localhost:3003
- **Polyglot Translator**: http://localhost:3004
